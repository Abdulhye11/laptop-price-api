# 💻Second hand Laptop Price Predictor in Pakistan

A machine learning web application that predicts the price of used laptops in Pakistan.

## Features

- Predict laptop prices instantly
- Streamlit user interface
- Scikit-learn Pipeline
- Supports multiple laptop specifications

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
